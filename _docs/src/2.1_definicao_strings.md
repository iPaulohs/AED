# Princípios básicos sobre Strings

### Definição de string
Uma cadeia de caracteres é um objeto do tipo `String` cujo valor é texto. Internamente, o texto é armazenado como uma coleção sequencial somente leitura de objetos `Char`. A propriedade `Length` de uma string representa o número de objetos Char que ela contém, não o número de caracteres Unicode. 

```csharp
string name = "John Doe";
var length = name.Length;
Console.WriteLine(length); //8
```

Para acessar os caracteres Unicode individuais, usamos o objeto `StringInfo`. 
```csharp
TextElementEnumerator charEnum = StringInfo.GetTextElementEnumerator(name);
while(charEnum.MoveNext())
{
    Console.WriteLine(charEnum.GetTextElement());
}
```

### string vs System.String
Em C#, a palavra-chave `string` é um alias para classe `String` e, portanto, as duas formas são equivalentes. O alias string funciona mesmo sem `using System`;. A classe String fornece muitos métodos para criar, manipular e comparar cadeias de caracteres com segurança.

### Interpolação de strings
A interpolação de strings funciona adicionando `$` antes das aspas ou com o operador `+` entre duas ou mais strings.

```csharp
string surname = "Doe";
string name = $"John {surname}";
Console.WriteLine(name); //John Doe
```
Ou
```csharp
string name = "John" + "Doe";
Console.WriteLine(name); //John Doe
```

### Strings são imutáveis
Em C#, strings são sempre imutáveis e não podem ser alterados depois de criados. Todos os métodos que parecem alterar uma string, na verdade, retornam uma nova string.

```csharp
var s1 = "In C#, strings are";
var s2 = "always immutable objects.";

var s3 = s1 += s2; //Nova string
Console.WriteLine(s2); //In C#, strings are always immutable objects.
```

Se houver a necessidade de alterar algum caractere específico, a string pode ser transformada em um array de char. Assim, é possível acesso e modificação de cada posição de forma individual. A conversão de `string` em `char[]` é feita com o metodo `string.ToCharArray()`.

```csharp
var s4 = "C# is the better lang";
var s4CharArray = s4.ToCharArray();
Console.WriteLine(s4CharArray.GetType()); //System.Char[]
```

Para alterar alguma posição, basta substitui-la.

```csharp
s4CharArray[1] = 'S';
Console.WriteLine(s4CharArray); //CS is the better lang
```
